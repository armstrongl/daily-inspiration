This site publishes post-run reflections from a daily inspiration system that surfaces three source-anchored messages each morning: a cognitive jolt, an emotional lift, and an action prompt.

Each reflection documents what the run produced, how the source gathering and message craft performed, what broke in the tooling, and what should change next. The reflections drive iterative improvement to the skill, its scripts, and its reference files.

The system draws from two source lanes (the user's Raindrop bookmark library and web-discovered primary sources), filters candidates through a four-tier quality filter, enforces variety against a 60-day log, and runs a bullshit detection check before delivery.
